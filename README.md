# Metadata Scrubber

Утилита для автоматического удаления метаданных из файлов при копировании.

## Особенности

- Очистка EXIF, GPS, IPTC и других метаданных
- Поддержка изображений и документов
- Интеграция с системным треем Windows
- Всплывающие уведомления
- Настройка типов удаляемых данных

## Технологии

- Python 3.10+
- Pillow - для работы с изображениями
- watchdog - для мониторинга буфера обмена
- pywin32 - для интеграции с Windows

## Установка

```bash
pip install poetry
poetry install
```

## Запуск

```bash
poetry run python src/main.py
```
