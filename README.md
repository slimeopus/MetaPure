# Metadata Scrubber

Утилита для автоматического удаления метаданных из файлов при копировании.

## Особенности

- Полная очистка EXIF, GPS, IPTC и других метаданных из изображений
- Поддержка различных форматов изображений (JPEG, PNG, TIFF и др.)
- Работа в фоновом режиме с иконкой в системном трее
- Гибкая настройка типов удаляемых метаданных
- Сохранение настроек между сеансами
- Просмотр статистики обработки файлов
- Отсутствие всплывающих уведомлений для минималистичного интерфейса

## Технологии

- Python 3.10+ - язык программирования
- Pillow - для работы с изображениями и удаления метаданных
- watchdog - для мониторинга буфера обмена в реальном времени
- pywin32 - для интеграции с Windows (системный трей)
- TOML - для хранения настроек пользователя

## Конфигурация

Приложение использует файл `config.json` для хранения настроек пользователя. Настройки сохраняются между сеансами и включают:

- Типы удаляемых метаданных (EXIF, GPS, IPTC и др.)
- Состояние защиты (включено/выключено)

## Работа с приложением

1. Запустите приложение с помощью команды выше
2. В системном трее появится иконка приложения
3. Нажмите на иконку для открытия интерфейса (или дважды кликните по ней)
4. Включите "Active Protection" для начала мониторинга буфера обмена
5. Скопируйте изображение или документ - метаданные будут автоматически удалены
6. Просматривайте статистику обработки в главном окне
7. Настройте типы удаляемых метаданных в соответствии с вашими предпочтениями
8. Используйте функцию Smart Preview для проверки изменений перед очисткой
